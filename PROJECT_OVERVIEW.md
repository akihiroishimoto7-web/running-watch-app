# PROJECT_OVERVIEW

## 目的

ランニング初心者〜サブ4ランナーが、自分に合うランニングウォッチを **1分** で診断できる Web アプリ。
最終的に「Garmin か Apple Watch か」 → 「具体モデル」まで迷わず辿り着くことをゴールにする。

公開URL: `https://runningwatchapps.netlify.app/`

## ターゲット

| セグメント | 状態 | 主な悩み |
| --- | --- | --- |
| ランニング初心者 | これから始める / 始めたばかり | 何を買えばいいか分からない |
| ジョガー | 月20〜60km、ゆるく走る | 走力よりライフスタイル重視 |
| サブ4挑戦層 | 月100〜200km、自己ベスト狙い | データで成長を追いたい |

40代男性のサブ4ランナーがコア。**怪しい広告感は出さない / 医療的・断定的表現は避ける**。

## Apple Watch と Garmin の役割の違い

| 観点 | Garmin | Apple Watch |
| --- | --- | --- |
| 思想 | ランニング特化 | 生活全体のスマートウォッチ |
| データ深度 | 圧倒的（VO2max, Training Readiness等） | 標準ワークアウトレベル |
| バッテリー | 11〜31日 | 18〜36時間 |
| 普段使い | 機能的だがランナー向け | デザイン・通知・決済すべて完結 |
| GPS精度 | マルチバンド対応機が多い | Ultra のみマルチバンド |
| 適性 | 走力を伸ばしたい人 | ランも生活も大事な人 |

## 取り扱いモデル（5機種）

1. **Garmin Forerunner 165** — 初心者向け / 約45,000円〜

4. **COROS PACE 3** — 軽量コスパ / 約38,500円
5. **Apple Watch** — 普段使い重視 / 約34,000〜130,000円超
